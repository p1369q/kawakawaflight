# かわかわふらいと

「かわかわふらいと」は、5〜10歳くらいの子どもが、空の工作室で自分だけの飛行機を組み立て、すぐにテストフライトで星集めを楽しめるブラウザゲームです。

## 使用技術

- TypeScript
- HTML Canvas
- CSS
- LocalStorage
- npm scripts

> 当初は Phaser 3 を使う構成を想定していますが、この環境では npm レジストリから依存取得が 403 で拒否されたため、MVPは外部依存なしのCanvas実装で完成させています。シーン、パーツデータ、保存、音、評価ロジックは将来 Phaser 3 へ移しやすいよう分離しています。

## インストール方法

```bash
npm install
```

このMVPは外部依存がないため、ネットワークが制限された環境でもそのまま動かせます。

## 開発サーバーの起動方法

```bash
npm run dev
```

ブラウザで `http://localhost:5173` を開きます。

## ビルド方法

```bash
npm run build
```

`dist/` に配布用ファイルが作られます。

## テスト方法

```bash
npm test
```

パーツ能力値、ランダム生成、保存データの正規化、評価、自己ベスト判定を確認します。

## 操作方法

- タイトル画面: 「はじめる」で工房へ進みます。
- 工房: 胴体、翼、エンジン、色を選び、「テストフライト」を押します。
- PC: Spaceキー、またはマウス長押しで上昇します。離すとゆっくり下降します。
- スマートフォン/タブレット: 画面を長押しで上昇、離すと下降します。
- 飛行中: 星と金の星を集め、エネルギーがなくなるか時間になると結果画面へ進みます。

## ディレクトリ構成

```text
src/
  main.ts                 ゲーム画面とCanvasループ
  game/
    data/planeParts.ts    パーツ、色、能力値計算

    systems/ResultSystem.ts 評価と結果生成
    systems/SaveSystem.ts LocalStorage保存
    types/game.ts         共有型定義
    utils/clamp.ts        数値制限
  ui/styles.css           レスポンシブUI
  test/runTests.mjs       ロジックテスト
scripts/
  build.mjs               依存なしビルド
  dev-server.mjs          簡易開発サーバー
```

## パーツ性能

### 胴体
- まるまるボディ: 操縦しやすいバランス型
- スピードボディ: はやさが高く、うきやすさは少し低め
- おおきなボディ: エネルギーが高く、はやさは少し低め

### 翼
- ひろびろウイング: うきやすさと操縦が高い
- スピードウイング: はやさが高く、うきやすさは少し低め
- ふわふわウイング: ゆっくり下降し、長く飛びやすい

### エンジン
- やさしいエンジン: 操作しやすい
- パワーエンジン: はやさが高く、エネルギーは少し減りやすい
- エコエンジン: 長く飛べるが、はやさは少し低め

## セーブデータ

LocalStorage の `kawakawa-flight-save` に次を保存します。

- 最後に選んだ胴体、翼、エンジン、色
- 最高飛行距離
- 最高獲得スター数
- 最高獲得金色スター数
- 音のオン/オフ
- 初回説明を見たかどうか
- セーブデータのバージョン番号

壊れた保存データやLocalStorageが使えない環境では、初期値またはメモリ保存にフォールバックします。

## 使用している素材とライセンス

画像、音声、3Dモデルなどの外部素材は使用していません。飛行機、星、雲、背景、効果音はコード内のCanvas描画とWeb Audio APIで生成しています。
